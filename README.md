# 🌌 星空小程序

一个基于 Vue3 + TypeScript + UniApp 开发的星空主题微信小程序，提供星座运势、实时星图、月相查询等功能。

## ✨ 功能特性

### 已实现功能

| 功能 | 描述 | 状态 |
|------|------|------|
| 🏠 首页 | 星空主题欢迎页、今日运势、功能入口 | ✅ 已完成 |
| 🔐 登录 | 微信登录、手机号登录 | ✅ 已完成 |
| 👤 我的 | 用户信息、收藏、设置 | ✅ 已完成 |
| 🧰 工具箱 | 星空工具分类浏览 | ✅ 已完成 |
| ♈ 星座运势 | 12星座每日运势、运势详解 | ✅ 已完成 |
| 🌌 实时星图 | 星座识别、AR模式（模拟） | ✅ 已完成 |
| 🌙 月相查询 | 月相日历、月出月落时间 | ✅ 已完成 |

### 界面特色

- 🎨 **星空主题设计** - 深蓝色渐变背景 + 闪烁星星动画 + 流星效果
- 💎 **毛玻璃效果** - 半透明卡片 +  backdrop-filter 模糊效果
- ✨ **发光效果** - 文字发光、按钮发光、月亮辉光
- 🌠 **动画效果** - 星星闪烁、流星划过、星云飘动

## 🛠️ 技术栈

- **框架**: Vue 3 + UniApp 3
- **语言**: TypeScript
- **样式**: 原生 CSS + 自定义主题
- **构建**: Vite

## 🚀 快速开始

### 环境要求

- Node.js >= 18
- 微信开发者工具

### 安装依赖

```bash
cd starry-sky-app
npm install
```

### 开发运行

```bash
# 运行到微信小程序
npm run dev:mp-weixin
```

### 构建发布

```bash
# 构建微信小程序
npm run build:mp-weixin
```

## 🔧 组件说明

### StarBackground 星空背景

全局背景组件，包含：
- 渐变背景
- 随机生成的星星（小/中/大三种尺寸）
- 流星动画
- 飘动的星云
- 发光的月亮

```vue
<StarBackground />
```

### GlassCard 毛玻璃卡片

全局样式类，用于创建毛玻璃效果卡片：

```html
<view class="glass-card">
  <!-- 卡片内容 -->
</view>
```
